Oncepkg

Version: 1.0.0

Install npm packages globally once and use them across all projects on your machine.

Features

  • Install a package globally once and use it in any project.
  • Automatically updates to the latest version when used in a new project.
  • List all globally installed packages via CLI.
  • Detects similar packages and prevents redundant installations.
  • Monitors npm command executions to keep track of installations.
  • Creates project structures for MERN and MEAN stacks with JavaScript or TypeScript.

Installation

npm install -g oncepkg

Usage

Install a package globally:

gpm install

List installed packages:

gpm --list

Scan old projects for missing packages:

gpm --scan

Create a project with a template:

gpm --template mern --js

Limitations

  • Requires global installation of the package.
  • Does not track manually installed packages outside of npm.
  • May not work well with system-wide installations or version conflicts.

Warnings & Security Considerations

  • Be cautious when installing packages globally; ensure they are from trusted sources.
  • Keep packages updated to avoid security vulnerabilities.
  • Check for redundant or conflicting packages before installation.
  • Use version locking in critical projects to prevent unexpected updates.
